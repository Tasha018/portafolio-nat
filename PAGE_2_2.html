Señaletica para Extintor

Este proyecto consistió en el diseño y fabricación de una señalética luminosa para la UAM Cuajimalpa, utilizando como herramienta principal la cortadora láser. Tras una investigación sobre la problemática, mi equipo y yo decidimos crear una estereotomía de un extintor, fabricada en acrílico transparente. Para su montaje, diseñamos una base de MDF, pensada para incrustarse en la pared. Además, esta base también funcionaba como soporte para la fuente de luz, la cual iluminaba la estereotomía, mejorando su visibilidad. Para simplificar la instalación y evitar la necesidad de un sistema de conexiones eléctricas, la luz fue diseñada para funcionar con pilas, garantizando así mayor flexibilidad y facilidad de uso.
